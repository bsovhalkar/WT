# Experiment 6 - Web Application Deployment

## Aim
Deploy a basic web application and ensure public accessibility with basic error handling.

## Included Files (as mentioned in manual examples)
- `index.html`
- `style.css`
- `script.js`
- `404.html`
- `app.py` (dynamic hosting sample with 404/500 handlers)

## Static Deployment (GitHub Pages)
Deploy `index.html`, `style.css`, `script.js`, and `404.html`.

## Dynamic Deployment (PythonAnywhere/Flask)
Upload `app.py` and configure WSGI entrypoint to this Flask app.
